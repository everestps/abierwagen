Hardware

Motherboard

This is a motherboard (well it is a picture that is graphically enhanced). The motherboard is the main part of the computer that hold all the valuable parts.

CPU

This is a CPU (Central Processing Unit). This part of the computer handles all of the calculations of the computer and all the heavy work.

RAM

This is 2 sticks of RAM (Random Access Memory). This part of the computer is the part where it stores programs that run on your computer's data.

HDD

This is the HDD (Hard Disk Drive). This is the storage part of your computer where big files go. Nowadays there are faster drives called SSD's (Solid State Drives) which dont have a spinning part.
