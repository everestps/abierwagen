What is HTML and why is it important

HTML is an acronym for Hypertext Markup Language that is interpreted by the browser but there are sometimes differences between the different browsers

The reason it is important is because it was created to layout webpages

IMHO we need a new standard for webpages since webapps are made with specific purpose and HTML doesn't acomplish it very well

What is CSS and why is it important?

CSS is an acronym for Cascading Stylesheets that is also interpreted by the browser but also there are differences between browsers

The reason CSS is so important is because it is the language that defines the style and structure values
