What is HTML and why is it important

HTML is an acronym for Hypertext Markup Language that is interpreted by the browser but there are sometimes differences between the different browsers.

The reason it is important is because it was created to layout webpages.

What is CSS and why is it important?

CSS is an acronym for Cascading Stylesheets that is also interpreted by the browser but also there are differences between browsers.

The reason CSS is so important is because it is the language that defines the style and structure values.






How do tags work?

Tags are the building blocks of HTML, they layout and define how the HTML is rendered.

The way tags work is they have the < to start the tag then the text goes inside and you can have attributes in the tag, then you close them with > so a finished tag would look like <code>code in here<code> would render as code in here

Description of at least 4 different tags

  • ul
    • Tag Name: Unordered list
    • You use this tag when you want to start of some bullet points
    • It's sister element is ol is used when you want a list with numbers.
  • li
    • Tag Name: List item
    • You use this tag when you want to have a list item
    • Must be nested inside a ul or ol
  • div
    • Tag Name: Divider
    • You use this tag when you want to have specific dividers between elements.
  • span
    • Tag Name: Span
    • You use this tag when you want to use a div but you are putting it inline inside a block element.
